Movie File Merger

The purpose of Movie File Merger (MFM) is to help you with the selective combination video collections, like movies, series, or documentaries. Some examples of what you can do with MFM are...

  • Select which videos are copied or moved
  • Select the videos to copy before hand
  • Copy only relevant videos in the Wish list
  • Ignore videos in the Existing and Garbage lists
  • Select movies according to predefined genere lists from IMDb
  • Select videos according to regular expressions; for example year 2013...
  • See video media information in the pop-ups when hovering over a list entry
  • Select videos according to filtered media information; for example horizontal resolution greater than 1000 pixels

MFM is not a movie database program!

MFM is designed so that the lists can be easily updated once out of date. MFM is designed to run in Dropbox, which makes it easier to share lists with other people, update the program, and store your settings and lists in a save place. MFM is designed to work together with multiple other programs, like GOM, VLC, MediaInfo, TeraCopy, All of them can be downloaded and used for free.
