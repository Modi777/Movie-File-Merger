Wikis

MFM is basically built up out of 3 separate GitHub projects. That's why it has also 3 wikis.

  • The Movie File Merger Wiki, which contains information related to the program, a wiki version of the user manual and how to organize videos combined.
  • The Movie File Merger User Manual Wiki, which contains some information related to the user manual GitBook.
  • The How to Organize Videos on Your Hard Disks Wiki, which contains some information related the how to organize videos GitBook.
